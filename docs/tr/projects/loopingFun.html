Eğlenceli Döngüler

Yazan: Lisa Yan

Bu egzersizde Java döngülerini pratik etmenizi saylayacak programlar verilmiştir. Programlar kolaydan zora doğru sıralı olarak verilmiştir.

Döngüleri Hatırlayalım

For döndüsünün genel yapısını hatırlayalım: 

                    for(int i = 0; i < 15; i++) {
                    // some code...
                    }
                

            
Yukarıdaki döngü 15 kez tekrarlanmaktadır. İlk satırdan döndünün ne anlama geldiğini çıkartabilirsiniz:

            
    
                
  • Sıfırdan başlayan i isminde bir sayaç yaratın.
    • 
                
  • Her turda i < 15 ifadesi doğru mu diye bakın.
    • 
                
  • Eğer i < 15 ise döngünün içindeki kodu çalıştırın. Döngü kodunu kullanırken i değişkeninin değerini kullanabilirsiniz. Örneğin döngünün ilk turunda i değişkeninin değer sıfırdır.
                
  • Döngü kodunun sonunda i++ ifadesi ile değişkenin değerini 1 arttırın.
    • 
                
  • Eğer !(i < 15) ise döngüden çıkın. Dolayısıyla döngü kodunun çalıştığı son turda i değişkeninin değeri 14'tür ve sonunda 15'e yükseltilecektir. Böylece i < 15  kontrolünden geçemeyecek ve kod çalışmayacaktır.
    • 
            

            

            
Dikkat! Döngü koşullarında programlama hatalarını ("bug") bulmak çok zordur! Dolayısıyla her zaman en basit döngüleri kullanmanızı ve sayacı 1 arttırmanızı (i++) öneririz. Ayrıca sayacın değerini döngü kodu içerisinde de değiştirmemeye önem gösterim!

            
    
                
  • Aşağıdaki kod sadece 4 defa çalışacaktır!!! Nedenini görebiliyor musunuz?
    • 
            
                    // Don't do this! This is hard to debug!
                for(int i = 0; i < 10; i ++) {
                    i += 2; // bad!!!
                    println(i);
                }
            
    
                    
  • Yukarıdaki işlemi yapmanın daha güzel bir yolu:
    • 
            
                // Uses another variable to keep the loop counter safe
            for(int i = 0; i < 10; i ++) {
                int j = i + 2; // does not change the value of i
                println(j);
            }
            
    
                

                

            

                Aşağıdaki egzersizlerin tamamı for döngüsünün sayacı i ile alıştırmaları kapsamaktadır.

Aşağıdaki çıktıyı bir döngü kullanarak oluşturmaya çalışın. ilk 10 çift sayıyı azalan şekilde yazdırın.

            

                	
            

            
Bir önceki döngüde kullandığınız koşu değiştirmeden bu problemi de çözebilir misiniz? Yani bu egzersiz için aşağıdaki kod ile başlayın:

            
            for(int i = 0; i < 10; i++) {
                    // some code...
            }
            

            
Peki bu problemi aşağıdak döngü koşulu ile nasıl çözerdiniz?

            
            for(int i = 0; i < 20; i++) {
                    // some code...
            }

Eğlenceli Döngüler 4

        
BonusConsole: LoopingFun4.java
            
Artık iç içe döngülere geçebiliriz! 5'e 5'lik bir haritanın koordinatlarını yazdırın.

            

                	
            

            
İpucu: Aşağıdaki kod yapısını deneyin:

            
            for(int i = 0; i < 5; i++) {
                    for(int j = 0; j < 5; j++) {
                            // some code...
                    }
            }
