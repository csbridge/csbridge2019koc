3. Rastgele Sayılar

Day2: RandomNumbers.java

0 ile 100 arasında 1000 tane rastgele sayı üretip bu sayıları ekrana yazdıran bir program yazın.

RandomNumbers.java dosyası içinde rastgele sayı üreten örnek kodu bulabilirsiniz. 

public class RandomNumbers extends ConsoleProgram {
							
						// A random number generator
						private RandomGenerator rgen = new RandomGenerator();

						public void run() {
							// change this code to print 1000 random numbers
							// in the range 0 to 100.	
							int example = rgen.nextInt(0, 10);
							println(example);	
						}
						}

Rastgele bir sayı üretmek için RandomGenerator değişkeni yaratın (tüm metotların dışında yaratılmış bir değişken):

private RandomGenerator rgen = new RandomGenerator();

Sonrasında bu değişkeni kullanarak yeni sayılar elde edebilirsiniz.

rgen.nextInt(min, max) //generates a random int in the range (min, max) inclusive
