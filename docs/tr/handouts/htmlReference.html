CSS ile Web Sayfanıza Tarz Katın

Başlangıç

HTML sayfınızın başında CSS dosyanızı da eklemeyi unutmayın:

<head>
  <link rel="stylesheet" type="text/css" href="styles.css">
</head>

Bu örnekteki styles.css CSS dosyasının adı. Sizin dosyanızın adı farklıysa kendi dosya adınızı yazmayı unutmayın

HTML'nizi CSS'e Hazırlamak

HTML elemanlarınıza class etiketini ekleyin. Mesela:

<p class="sınıf-adı">Merhaba!</p>

CSS'in Kendisini Eklemek

CSS dosyanızda, HTML elemanlarına eklediğiniz sınıs isimlerini kullanarak her sınıfa bir tarz yaratabilirsiniz:

.sınıf-adı {
  color: white;
  font-size: 20px;
  text-align: center;
  background-color: black;
}

Daha fazla öğrenmek isterseniz bir asistandan yardım isteyin.
