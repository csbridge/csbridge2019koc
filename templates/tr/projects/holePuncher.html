Ders Notları: Grafik Referansı Olaylar (Events) Referansı
5. Gün: HolePuncher.java

Bu örnek bilgisayarınıza bir kağıt parçası ve farenize de bir deleç gibi davranır. Kullanıcı istediği bir yere tıklar ve tıkladığı yerde delikler (GOval) açılır.

Açacağınız deliklerin yarıçapını belirten sabit değerler tanımlayın. Bu iş için 10 iyi bir değerdir. GOval objelerinin bu değer için double (int değil) türünde değişkenler istediğini unutmayın. Sonrasında bir fare dinleyicisi (mouseListener) yaratın ve her tıklama olayında çalışan draw isimli bir metot oluşturun.

Çözüm

/**
 * Class: HolePuncher
 * -----------------
 * A program that draws filled black circles on the screen whenever
 * the mouse
 * is pressed.
 */
public class HolePuncher extends GraphicsProgram {
  /* The radius of each hole that we punch. */
  private static final double HOLE_RADIUS = 10;

  public void run() {
    addMouseListeners();
  }

  /**
   * Draws a black circle centered at the mouse location whenever the mouse
   * is pressed.
   */
  public void mousePressed(MouseEvent e) {
    double x = e.getX() - HOLE_RADIUS;
    double y = e.getY() - HOLE_RADIUS;

    GOval hole = new GOval(x, y, 2 * HOLE_RADIUS, 2 * HOLE_RADIUS);
    hole.setFilled(true);
    add(hole);
  }
}
